Flap链上互动渔场

玩法介绍

养鱼、偷捕、起网，按渔获积分分奖池。

Flap渔场是一个“鱼塘经营 + 偷捕博弈 + 周期分红”的链上互动玩法。玩家通过购买鱼苗、投放鱼塘、增氧催熟、布置护网、发起偷捕，争夺每期奖池释放的 BNB。

默认鱼塘3
最大鱼塘8
捕捞期2小时/期
奖池分流90% / 10%

经营成长

买鱼苗、投塘位、扩鱼塘，决定你的产能上限。

攻防博弈

增氧抢时间，护网做防守，偷捕抢走别人部分积分收益。

周期分红

最后不是比鱼多，而是比积分高，积分越高分到的 BNB 越多。

参与流程

01默认拥有 3 个鱼塘
02购买鱼苗并投放空闲塘位
03增氧提速或布置护网
04也可对其他玩家发起偷捕
05成熟后起网，结算渔获积分
06期末按积分占比领取 BNB

奖池规则

资金来源

扩塘、增氧、护网与主动充值的 BNB，90% 进入渔场奖池，10% 进入运营池。

结算方式

每期释放本期渔池和滚存渔池各 50%，按个人积分占比分配。

常见问题 FAQ

Q：不花 BNB 能玩吗？

可以购买鱼苗和投苗，但如果不扩塘、不增氧、不布防，收益效率和防守能力会偏弱。

Q：积分可以提现吗？

积分本身不能提现，积分是分红凭证。捕捞期结束后，按积分占比领取 BNB。

Q：偷捕会偷走我的鱼吗？

不会直接偷走整条鱼，偷捕偷走的是你最终积分收益的一部分。

Q：护网有什么用？

护网可以降低被偷捕次数，尤其适合保护高价值鱼。

Q：为什么奖池不会一次分光？

因为系统每期只释放一部分，本期渔池和滚存渔池各释放 50%，剩余部分会滚入后续周期，保证后续持续有奖池。
